Aim: Database Modifications using DDL Commands

Analyze a system to implement the concepts of Database Modifications using Data Definition Language (DDL) Commands.

Recommended Hardware/Software Requirements:

  • Hardware Requirements: Intel Based desktop PC with minimum of 166 MHZ or faster processor with at least 64MB RAM and 100 MB free disk space.
  • MySQL 5.6.1

Pre-requisites:

Students must know DDL commands

DDL Commands:

  • CREATE - Used to create new database objects such as tables, views, indexes, etc.
  • ALTER - Used to modify the structure of existing database objects.
  • DROP - Used to delete database objects.
  • TRUNCATE - Used to remove all records from a table, but not its structure.
