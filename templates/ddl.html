This experiment focuses on Data Definition Language (DDL) commands in SQL. DDL commands allow us to define, modify, and manage the structure of the database. They include:

  • CREATE Command: Used to create new database objects such as tables, views, indexes, etc.
  • ALTER Command: Used to modify the structure of existing database objects.
  • DROP Command: Used to delete database objects.
  • TRUNCATE Command: Used to remove all records from a table, but not its structure.

Students will learn how to use these commands to define and manage the structure of a SQL database.
