This experiment focuses on Aggregate Functions in SQL. Aggregate functions perform calculations on a set of values and return a single value. Commonly used aggregate functions include:

  • COUNT: Returns the number of rows in a set.
  • SUM: Calculates the sum of values in a set.
  • AVG: Calculates the average value of a set.
  • MIN: Returns the minimum value in a set.
  • MAX: Returns the maximum value in a set.

Students will learn how to use these aggregate functions to perform data analysis and obtain useful insights from their SQL databases.
