Experiment 1: ER Diagrams

This experiment covers the basics of creating Entity-Relationship (ER) diagrams to represent the structure of a database.

Experiment 2: DML Commands

This experiment focuses on Data Manipulation Language (DML) commands such as INSERT, UPDATE, and DELETE to interact with the data in a database.

Experiment 3: DDL Commands

This experiment explores Data Definition Language (DDL) commands like CREATE, ALTER, and DROP to define and manage the structure of a database.

Experiment 4: Querying

This experiment introduces querying techniques using SQL to retrieve, filter, and manipulate data stored in a database.

Experiment 5: Aggregate Functions

This experiment covers the usage of aggregate functions in SQL to perform calculations on sets of data, such as SUM, AVG, MIN, and MAX.
