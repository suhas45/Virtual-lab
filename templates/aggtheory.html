Aggregate Functions Theory

Introduction

Aggregate functions in SQL are used to perform calculations on a set of values and return a single value as a result. Common aggregate functions include AVG, COUNT, MAX, MIN, and SUM.

Example Table

Employee ID Name Age Salary
1 John Doe 35 60000
2 Jane Smith 28 55000
3 Michael Johnson 40 70000

Examples

AGGREGATE FUNCTIONS

a) AVG

MySQL> SELECT AVG(age) FROM employees;

OUTPUT: 34.3333

b) COUNT

MySQL> SELECT COUNT(*) FROM employees;

OUTPUT: 3

c) MAX

MySQL> SELECT MAX(salary) FROM employees;

OUTPUT: 70000

d) MIN

MySQL> SELECT MIN(age) FROM employees;

OUTPUT: 28

e) SUM

MySQL> SELECT SUM(salary) FROM employees;

OUTPUT: 185000
