DDL Theory

CREATE DDL Command
The CREATE command is used to create new database objects such as tables, views, indexes, etc. in the database.

Syntax of CREATE Command:
            CREATE TABLE table_name (
                column1 datatype,
                column2 datatype,
                ...
            );


ALTER DDL Command
The ALTER command is used to modify the structure of existing database objects like tables, views, etc.

Syntax of ALTER Command:
            ALTER TABLE table_name
            ADD column_name datatype;


DROP DDL Command
The DROP command is used to delete existing database objects such as tables, views, indexes, etc.

Syntax of DROP Command:
            DROP TABLE table_name;


TRUNCATE DDL Command
The TRUNCATE command is used to remove all records from a table, but not its structure.

Syntax of TRUNCATE Command:
            TRUNCATE TABLE table_name;
