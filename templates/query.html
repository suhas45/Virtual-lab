This experiment focuses on advanced SQL querying techniques. Students will learn how to use various SQL operators and functions to perform complex queries, including:

  • ANY and ALL operators
  • IN operator
  • EXISTS and NOT EXISTS operators
  • UNION and INTERSECT set operators
  • Constraints such as PRIMARY KEY, FOREIGN KEY, UNIQUE, and CHECK constraints

Students will practice writing SQL queries involving these operators and constraints to retrieve, filter, and manipulate data from a database.
