Aggregate Functions:

  • AVG: Retrieve the average value of a column.
  • SUM: Retrieve the sum of all unique values in a column.
  • COUNT: Retrieve the count of a column.
  • MAX: Retrieve the maximum value of a column.
  • MIN: Retrieve the minimum value of a column.

Clauses:

  • ORDER BY: The ORDER BY clause sorts the results of a query in ascending or descending order.
  • GROUP BY: Sometimes we want to apply aggregate functions to groups of rows.
  • HAVING: HAVING is like a WHERE clause except that it applies to the results of a GROUP BY query.

Views:

A VIEW is a table whose rows are not explicitly stored in the database but are computed as needed from the view definition. A VIEW is a computed table by taking reference from base tables.
