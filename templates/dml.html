This experiment focuses on Data Manipulation Language (DML) commands in SQL. DML commands allow us to manipulate data in the database. They include:

  • SELECT Command
  • INSERT Command
  • UPDATE Command
  • DELETE Command

Students will learn how to use these commands to retrieve, add, update, and delete data in a SQL database.
