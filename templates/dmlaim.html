Aim: Database Modifications using DML Commands

Analyze a system to implement the concepts of Database Modifications using Data Manipulation Language (DML) Commands.

Recommended Hardware/Software Requirements:

  • Hardware Requirements: Intel Based desktop PC with minimum of 166 MHZ or faster processor with at least 1GB RAM and 500 MB free disk space.
  • MySQL 5.6.1

Pre-requisites: Students must know DML commands

Procedure:

  1. Analyze the system to identify data modification requirements.
  2. Use appropriate DML commands to implement the required modifications.
  3. Verify the correctness of the modifications made.
